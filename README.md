# Side Quest Week 6 — Fox Platformer with Sound & Physics

---

## Description

- Modified Week 6 Example 5 base code
- Added sound effects for:
  - Landing after a jump
  - Attacking enemies
  - Collecting leaves
  - Taking damage
- Bonus: screen shake triggers with hurt sound on damage

---

## Controls

- `A / ←` — Move left
- `D / →` — Move right
- `W / ↑` — Jump
- `Space` — Attack
- `R` — Restart
